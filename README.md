# STANCE — Wheel & Tyre Fitment Calculator

A live alloy wheel & tyre fitment calculator. Compare two setups and see poke,
inset, offset (ET), rolling radius, ride-height change and speedo error update in
real time, with a technical cross-section diagram.

## Run it
Open `index.html` in any browser. No build step, no dependencies — a single
self-contained file (HTML + CSS + JS inline).

## Notes
- Poke, inset, offset, ride-height and speedo figures are exact.
- Overall rolling diameter uses the standard formula (rim + 2× sidewall); it is
  **not** tyre-stretch-adjusted, so it won't match willtheyfit.com to the mm.
